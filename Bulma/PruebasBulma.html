Bulma CSS
  • Bulma is CSS framework build with SASS.
  • Was created by Jeremy Thomas in 2015.
  • The last update is 0.7.1 in Setember 21 of 2018.
  • The name bulma is inspirated from the Dragon Ball character.

SASS Definition

Is a CSS preprocessor, which adds special features such as variables, nested rules and mixins into regular CSS. The aim is to make the coding process simpler and more efficient.

Flexbox Definition

It was designed as a one-dimensional layout model, and as a method that can help distribute the space between the items in an interface and improve the alignment capabilities.

Responsive Definition

It is about resizing and placing the elements of the web in a way that adapts to the width of each one of the media for a better visualization and a better user experience. It is described because the designs (contents) and images are fluid and the code is used as CSS3 queries.
